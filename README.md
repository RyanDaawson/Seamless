# Seamless 🎶
Smooth, beat-matched transitions for your Spotify playlists.

## Features
- Paste Spotify playlist URL
- Fetch tracks + audio features (BPM, key, energy)
- Smooth crossfade playback (4–7s)
- Suggested next song for seamless transition
- Tailwind-powered UI

## Tech Stack
- Frontend: React + Vite + Tailwind
- Backend: Node.js + Express
- Auth & Data: Spotify Web API (OAuth with PKCE)
